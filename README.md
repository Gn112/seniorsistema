# Configuração do Projeto

## Requisitos

- **Python**
- **Django**
- **Git**

---

##  Clonagem do Repositório

```bash
git clone https://github.com/Gn112/seniorsistema.git
cd .\seniorsistema\
```

*Obs: Faça no terminal do Vscode
## Passo 1: Crie o Ambiente Virtual

```bash
python -m venv venv
pip install django
```

## Passo 2: Ative o ambiente 

```bash
.\venv\Scripts\Activate
```

## Passo 3: Rode o servidor 

```bash
python manage.py runserver
```
